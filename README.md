# 마실 · 홈 화면 (masil-home)

로그인 없는 지역 나들이 지도 **「마실」**의 **홈 화면**입니다.
Vue.js 3 (Vite) 기반 프론트엔드 단일 구조(정적 SPA)로 개발되었습니다.

## 이 프로젝트의 범위

이번 프로젝트는 **홈 화면만** 담당합니다.
지도 · 캘린더 · 커뮤니티 기능은 **각각 별도로 개발한 뒤 git 을 통해 홈 화면과 연결**할 예정입니다.
그래서 상단 탭과 하단 카드의 「지도 · 캘린더 · 커뮤니티」를 누르면 지금은 **준비 중** 화면으로 안내합니다.

### 홈 화면 구성

- 히어로 배너 캐러셀 (기본 슬라이드 + 서울 5개 권역 대표 명소)
- 서비스 소개 카드 (왜 마실을 만들었나 / 마실이란?)
- 핵심 통계 (등록된 장소 · 생활권 · 카테고리 · 필요 로그인 0)
- 카테고리별 장소 도넛 차트 (권역 필터 지원)
- 권역별 장소 막대 그래프
- 지도 · 캘린더 · 커뮤니티로 이동하는 내비게이션 카드

## 데이터

홈 화면 렌더링에 필요한 집계값(통계·도넛·권역·히어로 이미지)은
원본 장소 데이터에서 미리 추출해 [src/data/home-data.js](src/data/home-data.js) 에 정적 스냅샷으로 담았습니다.
(원본 3MB 데이터를 통째로 싣지 않아 가볍습니다. 지도 기능이 연결되면 실시간 집계로 교체할 수 있습니다.)

## 실행

```bash
npm install
npm run dev      # 개발 서버 (http://localhost:5174)
npm run build    # dist/ 로 정적 빌드
npm run preview  # 빌드 결과 미리보기
```
